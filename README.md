# Student Performance Analysis


## Project Overview

This project analyzes student demographic and academic factors using Python.

The project focuses on understanding student-related patterns using Exploratory Data Analysis (EDA).


## Dataset

The dataset contains information about students including:

- Age
- Gender
- Ethnicity
- Parental Education
- Study Time Weekly
- Absences
- Tutoring
- Parental Support
- Extracurricular Activities


## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn


## Analysis Performed

- Data Loading
- Data Cleaning
- Handling Missing Values
- Duplicate Checking
- Numerical and Categorical Feature Analysis
- Exploratory Data Analysis
- Data Visualization
- Correlation Analysis
- Feature Scaling


## Project Files

- student_performance.csv → Dataset
- student_performance_analysis.ipynb → Complete analysis notebook
- main.py → Python script for analysis


## Conclusion

The project explores student demographic and academic factors through data analysis and visualization techniques.
